# pc-calendar
基于fullcalendar开发的 pc端日历

显示中国节日和世界节日信息

holiday: {
   "2020-06-01": "假",
   "2020-06-02": "假",
   "2020-06-25": "班"
},

根据数据源，显示节假日和上班日
添加日程和删除日程功能
点击加载更多功能
